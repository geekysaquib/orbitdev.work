// What it does

One console for the work around the work.

Everything a developer juggles between commits — boards, hours, containers, mail — in a fast, keyboard-driven interface.

▸

One-click launch

Open a project's editor, backend, terminal, and dev server together — through a local agent that drives your real tools.

▤

Sprints board

Your live Zoho Sprints kanban — real status columns, types, assignees, and attachments — without opening Zoho.

◷

Time & billable hours

An Orbit focus timer plus logged and billable hours pulled straight from Sprints, grouped by project and person.

⬢

Docker control

See running containers and images, and export any image to a tarball — read live from your machine.

✉

Mail

A read-only Gmail inbox with a clean two-pane reader, categories, and auto-refresh — triage without a context switch.

⌘

Command palette

Jump to any project or action with ⌘K. The whole console is built to be driven from the keyboard.

// How it works

Connect once. Command everything.

Connect

Add your keys in Settings — Zoho, Gmail, Docker. They're stored per-account, and a lightweight local agent bridges to the tools on your machine.

Command

Launch projects, work your board, log hours, check containers, and read mail — all from one console, without tab-hopping.

Ship

Stay in flow. ORBIT keeps the periphery one keystroke away so your focus stays on the code.
